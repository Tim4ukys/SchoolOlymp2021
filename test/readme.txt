Специальная программа для быстрой проверки программы.

Для проверки нужно открыть test.exe(или test, если Вы используете Linux) через консоль(или терминал). Один аргумент который принимает программа - это имя .EXE файла, в котором само решение. Сам EXE файл должен лежать в каталоге, в котором есть папка "tests". В этой папке "tests" должны быть файлы "01" с текстом ввода и "01.a" с готовым решением для проверки.

Вот пример команды, с помощью которой нужно запускать прогу: "tests.exe gd.exe". gd.exe - скомпилированная программа с решением.
